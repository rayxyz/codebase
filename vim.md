# show line or not show line number
```
:set number or :set nu
:set nonumber or :set nonu
```

# comment block
```
1. ctl + v or d to cut
2. arrow to select
3. shift + i
4. #
5. Esc
```

# uncomment block
```
1. ctl + v
2. arrow to select
3. x
```

# copy block
```
1. ctl + v
2. arrow to select
3. y
```

# paste
```
p
```
